# Spotify-Clone
This is website that looks just like spotify. I built it using only HTML and CSS to practice my web design skills and make it look exactly like the real app. 
<br> <hr>

🚀Features <br>
<b>Beautiful Design</b>: It uses Spotify's dark theme, sidebar, and music cards. <br>
<b>Music Player Bar</b>: A fixed bar at bottom with play, pause, volume, and progress bar. <br>

💻Tech Stack <br>
<b> HTML5 </b>: To create the structure and sections of the website. <br>
<b> CSS3 </b>: To add colors, fonts, layouts, and make it look pretty. <br>
<b> Font Awesome </b>: Icons for navigation and controls. <br>
<b> Google Fonts </b>: Typography enhancements. <br>
